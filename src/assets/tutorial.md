# 核心理念

我们不提供“傻瓜式”的一键生成，而是提供一套**硬核**的底层架构。

真正的创作者需要的是灵活的工具，而不是被限制在固定的框架内。

# 沉浸式写作

为了让你更好地进入心流状态，我们提供了多种沉浸式模式：

1. **打字机模式**：让当前编辑的行始终保持在屏幕中央。
2. **长文模式**：将所有场景无缝拼接，连贯阅读。
3. **伪装模式**：上班摸鱼必备，一键伪装成枯燥的财务报告。
