# 实验室游戏创意存档

这里记录了为“实验室”模块构思的多个解压与灵感激发类小游戏，将逐个实现。

## 1. 灵感扭蛋机 / 剧情老虎机 (Inspiration Gacha)
*   **玩法**：界面是一个复古的老虎机或扭蛋机。拉动摇杆，屏幕会随机滚动并停在三个词条上，比如：**[人物特征] + [特殊场景] + [突发冲突]**（例如：“*一个晕血的刺客*” + “*在深夜的自助洗衣店*” + “*突然捡到一个会说话的钱包*”）。
*   **作用**：纯粹的脑洞生成器。当卡文或者不知道写什么短篇练习时，摇一摇，强行把不相关的元素组合在一起，激发荒诞或有趣的灵感。

## 2. 击碎“卡文” (Writer's Block Breaker)
*   **玩法**：经典的“打砖块”游戏（Arkanoid/Breakout）的变体。上方排列的砖块上写着作者的日常烦恼，比如 **“拖延症”、“逻辑漏洞”、“大纲卡壳”、“刷手机分心”、“完美主义”**。你控制底部的挡板，弹射一颗发光的“灵感小球”去击碎这些砖块。
*   **作用**：物理层面上发泄写不出来的挫败感。看着“拖延症”被小球砸得粉碎，有一种莫名的爽快和解压感。

## 3. 绝不回头 / 极限速写 (The No-Backspace Challenge)
*   **玩法**：一个硬核的打字挑战小游戏。设定 3 分钟或 5 分钟的倒计时，在这个特殊的输入框里，**退格键（Backspace）和删除键（Delete）被完全禁用**。无论你打错了什么字，或者写得多烂，都只能硬着头皮往下写。如果停止输入超过 5 秒，屏幕边缘会闪烁红光警告。
*   **作用**：专治“写一句话删半句”的完美主义。强制作者关闭内心的“自我审查”，进入纯粹的输出心流状态。

## 4. 键盘八音盒 / 盲打禅意 (Typing Zen)
*   **玩法**：这不是一个有输赢的游戏，而是一个视听玩具。在这个模式下，你敲击键盘上的每一个字母，都会发出悦耳的钢琴、古筝或八音盒的单音符，同时屏幕上会在随机位置绽放水彩晕染或粒子特效。
*   **作用**：极度放松。当写了很久逻辑严密的正文，大脑疲惫时，进来闭着眼睛随便敲击键盘，听听自己“弹奏”的无规律音乐，清空大脑。

## 5. 词汇接龙生存战 (Word Chain Survival)
*   **玩法**：屏幕中心出现一个词（比如“世界”），你必须在倒计时（比如 5 秒）内，输入一个以“界”开头的词（比如“界限”），接着再接“限制”……随着连击数增加，倒计时会越来越短，背景音乐也会越来越紧张。
*   **作用**：激活大脑的词汇库，在紧张刺激的快节奏中唤醒语言中枢，适合作为写作前的“热身运动”。
