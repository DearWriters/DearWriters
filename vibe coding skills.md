## 关于如何使用云同步：

1. 用github账户注册vercel，然后在vercel上新建project
2. 用github账号注册supabase
3. 在vercel的storage里选择create database，然后连接上supabase
4. 跟ai编程的软件说，已经做好了从github到vercel到supabase的连接，问它接下来需要在supabase里新建表，让它直接给出sql。
5. 在vercel-storage页面里可以找到你的supabase url和anon key，用它来启用supabase cloud sync。

以上任何一步不会，都可以问ai，它肯定比我教的清楚。我当时就是问着ai，一步一步摸索出来的。
当然，上面的步骤非常麻烦，而且还不稳定，所以最简单的办法就是下载到本地，然后 `npm install`，`npm run dev`。
另外我相信各位有能力的程序员可以把这个程序加上webdav或者s3同步的，我自己懒得做了。

## 关于 vibe coding 的小技巧

简而言之最重要的一句话：

**“不要修改代码”。**

然后就是，每次提需求要尽量多写，如果说不清楚就给ai举例子。把需求说清楚之后，要ai根据你的需求来生成可行性分析和改造方案，并且**“绝对不要修改代码”**。在和ai把需求理清楚之后，再让ai开始改造，这样的效率会高很多。

ai非常蠢，所以需要你给它提供思路。你的思路清晰、方向明确，ai就是你手脚麻利的员工；反之，如果你自己都不知道一个功能要怎么实现，那ai也帮不了你。
